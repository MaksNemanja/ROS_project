	En ce qui concerne le challenge 1, il suffit de lancer le launch file challenge1.launch qui va executer le projet, comme ce n'est pas specifier si on arette le robot a la fin du challenge ou pas, on a decidé d'areter le robot a l'entrée du couloir .
	
2) Challenge 2 :
	Pour le deuxiemme challenge, il suffit de lancer le launch file challenge2.launch et tout devrait bien fonctionner

3) Challenge 3 :
		Pour le deuxiemme challenge, il suffit de lancer le launch file challenge3.launch et tout devrait bien fonctionner
		
4 ) Si vous voulez l'enchainement du challenge 1 et 2 vous pouvez directement lancerle launch file challenge.launch, il peut passer tout les challenges depuis le debut jusqu'a la fin de la piste (le challenge des pilliers est exclu du fichier)


remarque :
	On a tout preparé dans les launch files, les positions de debut incluses. Cela c'est fait en positionnant notre robot dans la ou on veut le faire spawn, on a pris les coordonnées puis on les a mit dans le launch file.
